# Contributor Guide

`mwsync` welcomes contributions!  See below for more:

## How to Contribute

- Open an issue or discussion before large design changes in our issue tracker:
  [https://github.com/robla/mwsync/issues](https://github.com/robla/mwsync/issues)
- Update relevant docs in `docs/` when behavior changes.
- Run basic checks before submitting:

```bash
python3 -m py_compile mwsync.py catmgr.py ledecopy.py
```
